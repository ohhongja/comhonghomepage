# comhonghomepage

간단한 glassmorphism 스타일의 제조업 회사 홈페이지 예시입니다.

생성된 파일:
- [index.html](index.html)
- [styles.css](styles.css)

로컬에서 확인하려면 프로젝트 루트에서 아래 명령을 실행하세요:

```bash
# 간단한 로컬 서버 (Python)
python3 -m http.server 8000

# 브라우저에서 열기:
# http://localhost:8000/index.html
```
제조업을 하고 있는 회사 홈페이지를 glassmorphism 스타일로 회사연혁, 사업분야, 연락처 등으로 작성할 것
